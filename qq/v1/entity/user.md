| 字段             | 类型    | 描述                           |
| ---------------- | ------- | ------------------------------ |
| id               | String  | 用户ID                         |
| username         | String  | 用户名                         |
| avatar           | String  | 用户头像地址                   |
| bot              | Boolean | 是否为机器人                   |
| unionOpenId      | String  | 特殊关联应用的 openid          |
| unionUserAccount | String  | 机器人关联的互联应用的用户信息 |
